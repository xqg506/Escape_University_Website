Introduction

Escape from Uni is a 2D campus-themed adventure game built in Java using the LibGDX framework. In this game, the player is a university student and the goal is to catch the bus and escape campus within five minutes.

️Objective

To successfully escape, the player must complete the following tasks before time runs out:

  • Go to Ron Cooke Hub to retrieve the key card.
  • Travel back to Langwith College to collect the phone.
  • Find the bus waiting outside the university and escape before it departs.

Technical Design

The game uses a multi-screen architecture that separates different environments. The project follows object-oriented design principles and includes specialized manager classes to handle different aspects of gameplay, for example:

  • BuildingManager: Controls environment structures and collision detection.
  • AudioManager: Manages background music and sound effects.
  • Lighting: Creates dynamic light and shadow effects for nighttime gameplay.

Classes

We used CRC (Class–Responsibility–Collaboration) cards to model the game’s object-oriented structure.

Main

Responsibilities:
 - Start the game
- Create main Screen instances
- Initialize global resources (sound effects, fonts, etc.)

Collaborators: GameScreen, AudioManager

GameScreen

Responsibilities: - Manage the main game loop - Render the map - Update player and item states - Detect collisions

Collaborators: Player, Goose, BuildingManager, Collectable, Lighting

Player

Responsibilities: - Control player movement - Handle input processing - Collect items - Manage score

Collaborators: GameScreen, Goose, Collectable

Goose

Responsibilities: - Enemy implementation - Torch stealing logic - Escape and chasing behavior

Collaborators: GameScreen, Player

BuildingManager

Responsibilities: - Manage map buildings and obstacles - Provide collision detection

Collaborators: GameScreen, Player, Goose

Collectable

Responsibilities: - Define collectible item properties - Implement interaction logic

Collaborators: GameScreen, Player
